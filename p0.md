## CONTEXT

HYDRA is a modular OSINT data ingestion, correlation, and visualization platform. It aggregates 140+ data streams across 28 thematic tiers using 10 adapter types, normalizes all records to a universal schema, stores them across 6 storage engines, schedules ingestion via cadence-aware pipelines, and correlates cross-tier data for intelligence products.

This prompt produces the foundational project skeleton and all shared contracts. Every file generated here is an integration contract referenced by all subsequent layers.

Tech stack: Python 3.12+, FastAPI, Apache Airflow, Docker, Kubernetes. Async-first. Pydantic v2 for all models.

### src/[PASSWORD]/models/normalized.py — INTEGRATION CONTRACT

This is the universal data schema. Every adapter produces it. Every storage engine accepts it. Every correlation pipeline consumes it.

1=Geophysical & Seismic, 2=Atmospheric & Weather, 3=Space Weather & Solar, 4=Satellite Imagery & EO, 5=Economic & Financial, 6=Law Enforcement, 7=Public Health, 8=International Orgs, 9=EU & Eurostat, 10=Asia-Pacific Gov, 11=Americas Gov Non-US, 12=ME/Africa/Central Asia Gov, 13=US State & Local, 14=Arms & Defense Trade, 15=Conflict & Event Data, 16=Cyber Threat Intel, 17=Social Media & Web OSINT, 18=Aviation & Maritime, 19=Sanctions & Financial Intel, 20=NBC Threat, 21=Human Rights, 22=Astronomy & Astrophysics, 23=Space Situational Awareness, 24=Geoscience & Earth Systems, 25=Environmental & Climate, 26=Global Health & Epi, 27=Energy & Infrastructure, 28=National Portal Index

**AccessLevel**: string enum — green, yellow, blue, orange, red (per HYDRA accessibility legend)

**GeoPoint**: Pydantic model, GeoJSON Point with [lon, lat] or [lon, lat, alt]

**GeoGeometry**: Pydantic model, supports all GeoJSON geometry types (Point, LineString, Polygon, Multi*)

**SourceMeta**: Pydantic model with fields:
- source_name (str, required): human-readable source name
- source_url (str): base URL
- adapter_type (str, required): which adapter produced this record
- access_level (str): green/yellow/blue/orange/red
- fetch_timestamp (datetime, default UTC now)
- raw_format (str): original format
- api_version (str): if applicable
- rate_limit_remaining (int | None)

**NormalizedRecord**: Pydantic model with fields:
- stream_id (str, required): globally unique within tier, format "{source_slug}_{record_id}"
- tier (Tier enum, required)
- timestamp (datetime, required, always UTC — validator converts naive to UTC)
- geo (GeoGeometry | None): populated when spatially referenced
- payload (dict[str, Any]): domain-specific data, schema varies by tier/source
- source_meta (SourceMeta, required)
- raw_hash (str, required): xxhash64 hex digest (16 chars) of raw payload bytes — used for dedup
- ingested_at (datetime, default UTC now)
- confidence (float, 0.0-1.0, default 1.0): data quality score
- tags (list[str]): freeform tags for filtering/correlation

Validators:
- `timestamp`: if string, parse via fromisoformat; if naive, attach UTC
- `raw_hash`: must be exactly 16-char hex string (xxhash64)
